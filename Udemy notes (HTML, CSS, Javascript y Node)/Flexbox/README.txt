Una unica fila o columna, contenidos en una dimension, 
no dos como css grid. Colocaremos los elementos en 
una fila y se pasa a la fila de abajo de forma ordenada, 
pero las columnas no necesariamente encajan en su
alineacion. Flexbox esta pensada en manejar una dimension.

La distribucion con flexbox tiene un container y los 
elementos tienen su propio ancho, se distribuyen desde
main start hasta main end en el eje principal (hor)
y el eje cruzado es el vert. De arriba a abajo es 
cross start y abajo es cross end. El ancho es el main size
y el alto es el cross size. Completo se llama main container.
